# 📊 Data Logger: Sensor – SD Card

This project uses Arduino to collect sensor data (temperature and light) and store it with timestamps in CSV format.

## 🔧 Components

* Arduino Uno
* Temperature Sensor
* LDR Sensor
* SD Card Module

## ⚙️ Working

The sensors read environmental data and send it to Arduino. The Arduino processes the data and logs it with timestamps in CSV format, which can be stored on an SD card for later analysis.
